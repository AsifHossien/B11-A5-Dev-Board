Question-1: What are the different ways to select an element in the DOM?

There are as much as 5 different ways to select an element in the DOM such as :

  • 1. getElementById.
  • 2. getElementsByClassName.
  • 3. getElementsByTagName.
  • 4. querySelector.
  • 5. querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML allows us to see exactly what is in the HTML markup contained within a string, including elements like spacing, line breaks and formatting.innerText property sets or returns the text content as plain text of the specified node, and all its descendants whereas the innerHTML property gets and sets the plain text or HTML contents in the elements. Unlike innerText, inner HTML let us work with HTML rich text and does not automatically encode and decode text. For getting all the text from an element, including what's not visible, textContent is your go-to. It ignores CSS, so we get the raw text without any styling.

Question-3: What is event delegation in the DOM?

Event Delegation is a pattern to handle events efficiently. Instead of adding an event listener to each and every similar element, we can add an event listener to a parent element and call an event on a particular target using the .target property of the event object.

Question-4: What is event bubbling in the DOM?

Event bubbling is a type of DOM event propagation where the event first triggers on the innermost target element, and then successively triggers on the ancestors (parents) of the target element in the same nesting hierarchy till it reaches the outermost DOM element

Question-5: How do you create, add, and remove elements using JavaScript?

We can create an element by using document.createElement("") property. To add it we can use .appendChild() property. To remove an element we can use remove() property.
